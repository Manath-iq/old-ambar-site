Лендинг народного трактира «Старый Амбар» в **Нижнекамске** (демо для заказчика).

## Ключевые технические решения

1. **Scroll-scrub hero.** `.hero-track` высотой 480vh (420vh на мобиле), внутри sticky-stage.
   Видео качается fetch'ем как blob; `video.currentTime` догоняет прогресс ScrollTrigger
   через lerp (×0.16) в `gsap.ticker`. Три «главы» текста кросс-фейдятся по прогрессу
   (тайминги в `js/main.js` → массив `chapters`).
2. **Видео ОБЯЗАТЕЛЬНО перекодировать в all-intra**, иначе скраб залипает
   (в исходниках 1 keyframe на всё видео):
   `ffmpeg -i src.mp4 -c:v libx264 -g 1 -keyint_min 1 -crf 22 -preset slow -pix_fmt yuv420p -movflags +faststart -an out.mp4`
   (+ версия `-vf scale=768:-2` для мобил). Новое видео от заказчика класть в `_src/`, результат — в `video/`.
3. **Дизайн-система** (dark-luxe, скилл-пак в корне локально): фон `#0C0906`, янтарь `#E3A34B`,
   Playfair Display (900 — ворд-марк, Cyrillic) + Manrope. Никаких фиолетовых градиентов/неона.
   Ворд-марк «СТАРЫЙ / АМБАР» — всегда двухстрочный постер; маска строки `.h1-line` имеет
   padding-top .18em, чтобы не резать бревис «Й».
4. **Интерактив:** сценарии-аккордеон (4 панели), ховер-превью блюд за курсором (desktop),
   банкетный калькулятор (гости × тариф), FAQ, live-статусы («открыто до 01/02», «ланч сейчас»),
   формы с валидацией телефона (фронтенд-заглушки, бэкенда нет).
5. **Мобилка:** sticky-CTA внизу («Забронировать стол» + звонок), брейкпоинты 1180/1024/640.
6. Schema.org: Restaurant (с aggregateRating 4.3/634) + FAQPage. Favicon — data-URI SVG.

## Workflow

- Превью: `preview_start` с конфигом `ambar-static` (порт 8137). В фоновой вкладке превью
  rAF замирает — GSAP-анимации «зависают»; это артефакт панели, не бага сайта.
  Для проверки скраба: `ScrollTrigger.update(); for(...) gsap.ticker.tick()` в eval.
- Деплой: просто `git push origin main` — Pages подхватывает сам (~30–60 с).
  Проверка: `curl -s https://manath-iq.github.io/old-ambar-site/ | grep Нижнекамске`.
- Коммиты — на русском, с суффиксом `Co-Authored-By: Claude ...`.
- `gh` CLI НЕ установлен; ветка одна — `main` (PR-флоу не используется).
