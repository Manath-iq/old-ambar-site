# «Старый Амбар» — контент-лист для продакшена
### Все изображения, видео, 3D и интерактивные виджеты, которые нужно снять/заказать перед запуском

Сейчас на сайте стоят **временные фото с Unsplash** (лежат в `assets/img/`) — они держат вёрстку,
но не передают реальную атмосферу трактира. Ниже — полный список того, чем их заменить,
с требованиями к съёмке. Приоритет: 🔴 критично для запуска · 🟡 желательно · 🟢 после запуска.

**Общие требования к съёмке.** Один стиль на весь сайт: тёплый приглушённый свет (лампы,
свечи, угли — 2700–3200K), тёмный фон (дерево, камень, текстиль), никакой вспышки «в лоб»
и белых столешниц. Формат: исходники 4000px+ по длинной стороне, на сайт — WebP/AVIF,
q≈75. Все фото — горизонтальные, если не указано иное.

## 1. Hero-видео 🔴

Файл: заменяет `assets/hero-scrub-1440.mp4` / `assets/hero-scrub-768.mp4`

Текущее видео (современная вилла) не про трактир. Нужен ролик 10–15 секунд,
который рассказывает историю по мере скролла — у нас три «главы»:

| Прогресс | Что в кадре |
|---|---|
| 0–35% | Общий план зала: дерево, своды, тёплый свет, самовары |
| 35–70% | Середина: кухня/подача — шашлык с углей, пар от солянки, руки, наливающееся пиво |
| 70–100% | Финал: накрытый стол крупно, камера медленно наезжает — «стол уже накрыт» |

Технические требования (критично для скраб-эффекта):
- медленное, равномерное движение камеры (слайдер/стедикам), без резких склеек;
- 24–30 fps, без ручных тряски и перепадов экспозиции;
- **перекодировать в all-intra**: `ffmpeg -i src.mp4 -vf scale=1440:-2 -c:v libx264 -g 1 -keyint_min 1 -crf 23 -pix_fmt yuv420p -movflags +faststart -an hero-scrub-1440.mp4` (+ версия 768px для мобильных);
- без звука (он всё равно muted).

Альтернатива, если видео пока нет: серия из 60–120 фото со слайдера («стоп-моушн» наезд на стол) — соберётся в тот же mp4.

## 2. Сценарии — 5 фото 🔴

Файлы: `scen-lunch.jpg`, `scen-family.jpg`, `scen-banquet.jpg`, `scen-sport.jpg`, `scen-delivery.jpg`
Формат: вертикальная доминанта в кадре приветствуется — панель раскрывается широко, но в свёрнутом виде видна узкая полоса. Главный объект — по центру кадра.

1. **Ланч** — стол офисной компании в обед: суп + второе + компот, дневной свет из окна.
2. **Семья/друзья** — большой стол, 6–8 человек, смех, «передай хлеб» — живая сцена, не постановка «в камеру».
3. **Банкет** — накрытый банкетный зал ДО прихода гостей: длинный стол, свечи, отглаженный текстиль.
4. **Матч** — зал во время трансляции: экран в глубине, болельщики с пивом, эмоция гола.
5. **Доставка** — фирменные пакеты/боксы «Старый Амбар» на пороге квартиры или в руках курьера. Важно: брендированная упаковка в кадре.

## 3. Хиты меню — 6 фото блюд 🔴

Файлы: `menu-shashlik.jpg`, `menu-pelmeni.jpg`, `menu-solyanka.jpg`, `menu-rulka.jpg`, `menu-lagman.jpg`, `menu-blini.jpg`
Формат: квадратные кропы 1:1 должны выживать (фото показывается и маленьким квадратом 84×84, и превью 250×170).

Единая схема: блюдо на тёмной посуде/доске, тёмный деревянный стол, боковой тёплый свет, пар/дымок где уместен. Обязательно реальные блюда из меню, а не фуд-стилизация «как в рекламе» — гость должен получить то, что видел.

- Шашлык — на шампурах, с углями/мангалом в расфокусе.
- Пельмени — в глиняной миске со сметаной, ложка в кадре.
- Солянка — крупно, с лимоном и маслинами, пар.
- Рулька — целиком на доске, нож воткнут, к ней пиво (связка с блоком пива).
- Лагман — «тянутая лапша на вилке» над пиалой.
- Блины — стопка + икра/сметана, «кружевной» край крупно.

## 4. Пиво 🔴

Файл: `beer.jpg` (+ 🟡 короткое видео 5–8 с для замены на `<video>`)

Кружка с логотипом «Старый Амбар» — **обязательно брендированная посуда**: пена переливается через край, подсветка сзади-сбоку (янтарное свечение), тёмный фон бара. Видео-вариант: замедленный налив из крана, 120fps slow-mo.

🟢 **3D-идея:** фотограмметрия фирменной кружки → интерактивная 3D-модель (three.js / model-viewer), вращается при скролле в секции пива. Эффектно и недорого, если есть предмет.

## Интерактив и виджеты (по приоритету)

### 🔴 Виджет брони столов
Сейчас форма — фронтенд-заглушка (валидация есть, отправки нет). Подключить одно из:
- **Remarked / Hostes / SmartReserve** — готовые виджеты бронирования для ресторанов РФ;
- или свой бэкенд: форма POST → CRM (Битрикс24/amoCRM) + подтверждение в WhatsApp через WABA.
Обязательно: цель в Яндекс.Метрике на отправку.

### 🔴 Виджет Яндекс.Карт (секция «Адреса»)
Сейчас — стилизованная схема-плейсхолдер. Заменить на JS API Яндекс.Карт с кастомной тёмной темой (`customization`), янтарные метки под палитру сайта, попапы с телефоном и кнопкой «Маршрут». Ключ API бесплатен до 25k показов/сутки.

### 🔴 Заявка на банкет → WhatsApp/CRM
Калькулятор готов (гости × меню = смета), нужно только принять лид: телефон + дата + выбранные параметры уходят менеджеру. Тарифы меню (1500/2200/3000 ₽) — заменить на реальные.

### 🟡 Афиша трансляций
Сейчас 3 статичные строки. Варианты: (а) редактируются руками в админке/JSON-файле; (б) автоматически из API-Football / Sports.ru RSS по командам «Рубин», «Ак Барс», сборная. Плюс кнопка «Стол на матч» с предзаполненным филиалом.

### 🟡 Виджет отзывов Яндекс.Карт
Официальный embed-виджет отзывов организации (iframe) вместо статичных цитат — это и доверие, и SEO-сигнал. По одному на филиал или общий рейтинг сети. Текущие три цитаты — плейсхолдеры, заменить реальными до запуска.

### 🟡 Ланч-меню дня
Кнопка «Ланч-меню на сегодня» пока ведёт в никуда. Минимум: PDF/картинка, обновляемая раз в день. Лучше: JSON + рендер прямо в секции ланча («сегодня: борщ, котлета по-домашнему, компот»).

### 🟢 3D / WOW-слой (после запуска)
- 3D-кружка фирменного пива (см. п. 4) — скролл-вращение в секции пива;
- панорамы залов 360° (Matterport или сферы в Я.Картах) — вкладка в секции «Атмосфера», очень конвертит банкетчиков «посмотреть зал не выезжая»;
- cursor-parallax на фото атмосферы (лёгкий слой, уже заложен hover-зум).
